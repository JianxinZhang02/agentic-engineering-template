# Changelog

## 0.1.0 - 2026-07-02

- 初始化 Agentic Engineering Git 仓库模板。
- 增加轻量 `AGENTS.md` 入口规则。
- 增加 `context/` 团队、项目、经验三层知识结构。
- 增加 `requirements/` 需求生命周期模板。
- 增加基础文档模板、4 个 Skill、5 个 Agent、5 个 command。
- 增加 Claude Code、Codex、WorkBuddy 工具适配说明。
- 增加人工 eval 用例和一个示例需求。

## 0.2.0 - 2026-07-02

- 增加 AI Policy、工具治理、指标规范。
- 增加分层 Prompt、规则切片、Hook 和 MCP 配置样例。
- 增加安全审查、验证循环、工具治理 Skill。
- 增加 verify、security-review、context-garden、tool-stocktake、doctor 标准动作。
- 增加 Prompt 回归、安全策略 eval 和 eval 指标说明。
- 增强需求模板中的 AI 参与说明、安全门禁、验证循环和指标记录。

## 0.3.0 - 2026-07-02

- 借鉴 Chromium `agents/` 增加核心 Skill 清单 `core_skills.json`。
- 增加 `agent-projects/`，用于长期 AI 工程计划。
- 增加 `prompts/knowledge_base.md`，形成任务到文档的检索路由。
- 增加 `eval/tasks/` 可复现任务级 eval 结构。
- 增加 prompt `.tmpl.md` 同步约定。
- 增加 `scripts/doctor.py`，机械化检查模板健康状态。
- 增加 `mcp-configs/extensions.catalog.json`，记录 MCP/扩展 owner、状态和风险。
