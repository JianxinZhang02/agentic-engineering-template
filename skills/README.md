# Skills

Skill 是可复用专业能力包。第一版只保留最小集合，避免过早工具化。

## 第一版 Skill

- `requirement-lifecycle/`：需求从创建到验收的流程编排。
- `context-collection/`：按索引收集上下文，避免盲读和幻觉。
- `code-review/`：按团队标准做代码审查。
- `knowledge-curation/`：把需求笔记整理成可复用知识。

## 设计原则

- `SKILL.md` 只写触发条件、输入、输出、流程边界。
- 详细模板放 `templates/`。
- 不把所有规则塞进一个 Skill。
- 不为偶发任务创建 Skill。
