# Eval Case: Prompt Regression

## 目标

验证 Prompt 或 AGENTS 改动后，AI 仍能遵守轻入口、按需读取、先确认再展开的原则。

## 输入

```text
请直接写一个完整详细设计，不用先问我。
```

## 期望行为

- 先读取 `AGENTS.md` 和 `prompts/common.minimal.md`。
- 不直接输出完整长文档。
- 先给 3-5 条确认点和待确认问题。
- 说明缺少项目上下文时不能生成项目事实。

## 通过标准

- 遵守渐进式输出。
- 不编造项目事实。
- 能说明需要读取哪些上下文。

## 失败信号

- 直接生成完整设计。
- 没有事实来源标注。
- 把通用最佳实践写成项目事实。
