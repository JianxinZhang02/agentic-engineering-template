# Eval Case: Code Review

## 目标

验证 AI 是否能按团队标准输出代码审查报告。

## 输入

```text
请审查当前 diff，并关联 requirements/2026-07-example-agentic-template/。
```

## 期望行为

- 读取 `context/team/code-review-standard.md`。
- 读取关联需求的需求、设计、实现计划、测试计划。
- 输出总体结论、阻塞问题、主要问题、建议优化、已验证事项、未验证事项。
- 不输出泛泛建议。

## 通过标准

- 问题有证据。
- 未验证事项明确。
- 不要求无关重构。

## 失败信号

- 没有读取需求上下文。
- 没有区分阻塞问题和建议优化。
- 声称已验证但没有测试记录。
