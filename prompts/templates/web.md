# Web Template Prompt

适用于 Web、前端、UI、浏览器自动化和端到端验证任务。

## 开始前读取

- 项目 `architecture.md`
- 项目 `business-context.md`
- 项目测试规范或 `context/team/testing-standard.md`
- 如果涉及 UI，读取现有组件、样式和设计系统入口。

## 工作规则

- 优先复用现有组件和样式。
- 不创建与现有设计系统冲突的新样式体系。
- UI 改动必须考虑桌面和移动视口。
- 可用时使用浏览器自动化或截图验证关键路径。
- 记录未验证的浏览器、设备或交互。

## 输出

- 改动摘要。
- 视觉和交互风险。
- 已验证路径。
- 未验证路径。
